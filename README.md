## 프로젝트 개요
코로나 장기화로 인한 재택근무의 증가로 비대면 업무에 적합한 그룹웨어의 필요성이 증가하고 있습니다. 원활한 의사소통과 정보 공유, 신속한 의사 결정이 이루어질 수 있는 업무 환경을 제공하며 편리하게 이용할 수 있는 그룹웨어 시스템을 만들자는 목표로 프로젝트를 기획하였습니다.
